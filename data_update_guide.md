# 상식 데이터 자동 업데이트 방법 선택 가이드

## 3가지 추천 방법 비교

### 🥇 방법 1: Google Sheets (가장 추천!)
**난이도**: ⭐ (매우 쉬움)  
**자동화**: ⭐⭐ (반자동 - 수동 입력, 자동 동기화)  
**비용**: 무료

**장점**:
- 설정이 가장 간단함
- 엑셀처럼 직관적으로 데이터 관리
- 언제든 바로 수정 가능
- 팀 협업 가능 (여러 명이 함께 상식 입력)
- 별도 서버 불필요

**단점**:
- 매일 수동으로 상식 입력 필요
- (하지만 한 달치 미리 작성 가능)

**구현 방법**:
1. Google Sheets 생성 → 상식 데이터 입력
2. "웹에 게시" → CSV 형식 선택
3. URL을 앱 코드에 넣기
4. 앱이 자동으로 최신 데이터 가져옴

**이런 분께 추천**:
- 프로그래밍 초보자
- 직접 상식을 선정하고 싶은 분
- 빠르게 MVP를 만들고 싶은 분

---

### 🥈 방법 2: Wikipedia API + GitHub Actions
**난이도**: ⭐⭐⭐ (중간)  
**자동화**: ⭐⭐⭐⭐⭐ (완전 자동)  
**비용**: 무료

**장점**:
- 완전 자동화 (손 안댐)
- Wikipedia의 "오늘의 역사" 자동 가져오기
- 신뢰할 수 있는 데이터
- GitHub Actions로 매일 자정 자동 실행

**단점**:
- GitHub Actions 설정 필요
- Wikipedia 데이터를 상식 형태로 가공 필요
- 역사 위주 (다양한 카테고리 어려움)

**구현 방법**:
1. Wikipedia API로 오늘의 역사 가져오는 스크립트 작성
2. GitHub Actions 워크플로우 설정
3. 매일 자정에 자동으로 실행되어 JSON 파일 업데이트
4. 앱이 업데이트된 JSON 사용

**이런 분께 추천**:
- 완전 자동화를 원하는 분
- GitHub 사용에 익숙한 분
- 역사 상식 위주로 만들고 싶은 분

---

### 🥉 방법 3: 혼합 방식 (Google Sheets + 백업 JSON)
**난이도**: ⭐⭐ (쉬움)  
**자동화**: ⭐⭐⭐ (반자동)  
**비용**: 무료

**장점**:
- Google Sheets의 편리함 + 안정성
- Sheets가 안 되면 로컬 JSON 사용 (폴백)
- 유연한 데이터 관리

**단점**:
- 약간 복잡한 로직 필요

**구현 방법**:
1. Google Sheets를 1차 데이터 소스로 사용
2. 로컬에 백업 JSON 파일 보관
3. Sheets 로딩 실패 시 로컬 JSON 사용

---

## 단계별 실행 플랜

### 단계 1: MVP (1-2일)
**Google Sheets 방식 사용**
- 10-20개 상식 직접 입력
- 날짜별 로테이션으로 매일 다른 상식 보여주기
- 일단 돌아가는 앱 완성

### 단계 2: 콘텐츠 확장 (1주)
**더 많은 상식 추가**
- 100개 이상 상식 데이터 구축
- 카테고리별 균형있게 배분
- 친구들에게 도움 요청 가능

### 단계 3: 자동화 (선택사항)
**GitHub Actions 추가**
- Wikipedia API 자동 크롤링
- 매일 새로운 역사 상식 추가
- 기존 Sheets 데이터와 병합

## 추천 시작 방법

### 🚀 빠른 시작 (오늘 바로!)
1. Google Sheets 생성
2. 아래 템플릿 복사 사용
3. 10개 정도 상식 입력
4. 커서로 코딩 시작

### Google Sheets 템플릿
```
스프레드시트 열 구조:
A열: id (1, 2, 3...)
B열: category (과학, 역사, 생활 등)
C열: emoji (🔬, 🏛️, 💡 등)
D열: title (상식 제목)
E열: summary (2-3줄 요약)
F열: detail (상세 설명)
G열: relatedIds (2;5;8 형식)
H열: date (2025-11-03 형식)
```

### 예시 데이터 5개
```
1, 과학, 🔬, 물은 왜 투명할까?, 물 분자는 가시광선을 흡수하지 않아 투명하게 보입니다., 물 분자의 구조가 가시광선 영역의 빛을 흡수하지 않고..., 2;5, 2025-11-03
2, 역사, 🏛️, 한글 창제 이유, 세종대왕이 백성을 위해 쉬운 문자를 만들었습니다., 1443년 세종대왕은 백성들이 한자를 읽지 못해..., 1;8, 2025-11-03
3, 생활, 💡, 세탁 표시 보는 법, 옷의 세탁 라벨로 올바른 세탁 방법을 알 수 있습니다., 세탁 라벨의 기호들은 국제적으로 통용되는..., 5;7, 2025-11-04
4, 문화, 🎨, K-POP의 시작, 서태지와 아이들이 한국 대중음악을 혁신했습니다., 1992년 서태지와 아이들의 등장으로..., 9, 2025-11-04
5, 건강, 🏥, 물 하루 권장량, 성인은 하루 약 2L의 물을 마시는 것이 좋습니다., 체중과 활동량에 따라 다르지만 일반적으로..., 1;3, 2025-11-05
```

3. **공공데이터 포털**

   - https://www.data.go.kr

   - "일상생활에 유용한 생활 상식 20개 알려줘"
